# syncatom

sync atom config easy.

## Installation

``` bash
$ npm install -g syncatom
```


## Usage

``` bash
$ syncatom push [reop]
# eg: syncatom push git@github.com:forsigner/vimer-atom.git

$ syncatom pull [reop]
# eg: syncatom pull git@github.com:forsigner/vimer-atom.git
```
